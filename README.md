UPI Payment Simulation System (Spring Boot)


This project simulates a UPI-based digital payment system using Java and Spring Boot. It mimics real-world payment workflows such as transferring money, validating users, handling failures, and tracking transaction history.

The system is designed to demonstrate backend development concepts and fintech transaction logic in a simple and understandable way.


🔹 Simulated UPI ID-based transactions (e.g., alice@upi)
🔹 Balance validation before payment
🔹 Success & failure handling:
Insufficient balance
Invalid UPI ID
🔹 Unique Transaction ID generation
🔹 Transaction history tracking
🔹 REST APIs for payment and data retrieval

Learning Outcomes
Implemented real-world payment workflow logic
Gained understanding of REST API development using Spring Boot
Applied Object-Oriented Programming (OOP) concepts
Learned handling of transaction states and error scenarios
